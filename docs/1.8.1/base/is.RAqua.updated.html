is.RAqua.updated {base}R Documentation

Checks for newer versions of RAqua on CRAN

Description

This function is intended to check for new versions of RAqua on CRAN.

Usage

is.RAqua.updated()

Details

If a new version of RAqua is found on CRAN, the user is asked about visiting CRAN for downloading the new version. If so, the corresponding CRAN url is opened in the favorite browser and no other action is taken. Is up to the user the final decision to download and install the new version of RAqua.
