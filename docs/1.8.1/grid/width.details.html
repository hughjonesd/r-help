width.details {grid}R Documentation

Width of a Grob

Description

This generic function is called during the evaluation of "grobwidth" units. It should return an object of class "unit".

Usage

width.details(x)

Arguments

x

A graphical object list structure.

Value

An object of class "unit".

Author(s)

Paul Murrell
