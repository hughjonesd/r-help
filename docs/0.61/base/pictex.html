pictex {base}R Documentation

A PicTeX Graphics Driver

Description

This function produces graphics suitable for inclusion in TeX and LaTeX documents.

Usage

pictex(file = "Rplots.tex", width = 5, height = 4, debug = FALSE,
       bg = "white", fg = "black")

Arguments

file

the file where output will appear.

width

The width of the plot in inches.

height

the height of the plot in inches.

debug

should debugging information be printed.

bg

the background color for the plot.

fg

the foreground color for the plot.

Note

Examples

\documentclass{article}
\usepackage{pictex}
\begin{document}
\begin{figure}[h]
  \centerline{\input{Rplots.tex}}
  \caption{}
\end{figure}
\end{document}
