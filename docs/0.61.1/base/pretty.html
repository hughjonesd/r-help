Pretty Breakpoints

Description

pretty returns a sequence of about n equally spaced nice values which cover the range of the values in x. The values are chosen so that they are 1, 2 or 5 times a power of 10.

Usage

pretty(x, n=5)

Examples

pretty(1:15)
pretty(1:15 * 2)
pretty(1:20)
pretty(1:20, n=2)
pretty(1:20, n=10)
