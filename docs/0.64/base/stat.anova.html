stat.anova {base}R Documentation

GLM Anova Statistics

Description

This is a utility function, used in anova.glm(..., test != NULL) and should not be used by the average user.

Usage

stat.anova(table, test, scale, df.scale, n)

Arguments

table

numeric matrix as results from anova.glm(..., test=NULL).

test

a character, matching one of "Chisq", "F" or "Cp".

scale

a weighted residual sum of squares.

df.scale

degrees of freedom corresponding to scale.

n

number of observations.

Value

A matrix which is the original table, augmented by a column of test statistics, depending on the test argument.

See Also

anova.glm.

Examples

##-- Continued from  ``?glm'':

print(ag <- anova(glm.D93))
stat.anova(ag$table, test = "Cp",
	scale= sum(resid(glm.D93, "pearson")^2)/4, df=4, n = 9)
