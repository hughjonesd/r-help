newEnv {nls}R Documentation

Create an environment

Description

newEnv returns a new (empty) environment enclosed in the enclos environment.

Usage

newEnv(enclos)

Arguments

enclos

An optional enclosing environment for the environment to be created. If enclos is missing the new environment will be enclosed in the parent's environment.

Value

A new (empty) environment.

WARNING

This function is deprecated and will be removed in release 1.2.0. Use new.env instead.

Author(s)

Saikat DebRoy <saikat@stat.wisc.edu>

new.env

Examples

str(myenv)
