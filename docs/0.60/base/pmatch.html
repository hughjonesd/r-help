pmatch {base}R Documentation

Partial String Matching

Description

pmatch seeks matches for the elements of its first argument among those of its second. If there is a unique match then the index of the matching value is returned; if multiple matches are found then 0 is returned and if no match is found then NA is returned.

This function is based on Terry Therneau's charmatch S function. In R, charmatch is synonym to pmatch.

Usage

   pmatch(x, table, nomatch = NA)
charmatch(x, table, nomatch = NA)

Arguments

x

the values to be matched.

table

the values to be matched against.

nomatch

the value returned at non-matching positions.

See Also

match.

Examples

pmatch("m",   c("mean", "median", "mode")) # returns 0
pmatch("med", c("mean", "median", "mode")) # returns 2
