savePlot {fixed}R Documentation

Save Windows Plot to a File

Description

Saves the current plot on a windows device to a file.

Usage

savePlot(filename="Rplot", type=c("wmf", "gif", "ps"),
         device=dev.cur())

Arguments

filename

The filename under which to save the plot, without the extension.

type

The type of plot, Windows metafile, GIF or PostScript.

device

A device number of a windows device, by default the current device.

Details

This is equivalent to selecting the ‘Save as’ menu item on the ‘File’ menu of a devga device. Although dev.print can be used for metafile and PostScript plots, this is the only way to obtain a GIF file (directly).

Value

None, but a plot file will be created.
