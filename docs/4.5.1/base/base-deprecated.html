Deprecated Functions in Package base

Description

These functions are provided for compatibility with older versions of R only, and may be defunct as soon as the next release.

The 12 as.data.frame methods listed here have been informally deprecated already for R 4.3.z (via environment variable _R_CHECK_AS_DATA_FRAME_EXPLICIT_METHOD_, but also in package checks) and formally from R 4.4.0. They are replaced typically by as.data.frame() or possibly as.data.frame.vector().

Details

Deprecated functions in standard packages other than the base package are listed in help("pkg-deprecated"), where pkg is replaced by the name of the package.
