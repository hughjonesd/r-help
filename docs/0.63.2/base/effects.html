Effects from Fitted Model

Description

Returns (orthogonal) effects from a fitted (generalized) regression model. In case of a full rank n \times p model.matrix, the first p effects correspond to coefficients and the remaining n-p ones to residuals.

Usage

effects(object, ...)
effects.lm(object, set.sign=FALSE)

Arguments

object

list; typically, the result of a model fitting function such as lm
