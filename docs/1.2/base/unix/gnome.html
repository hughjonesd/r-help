GNOME Desktop Graphics Device

Description

gnome starts a GNOME compatible device driver. ‘GNOME’ is an acronym for GNU Network Object Model Environment. gnome(display="", width=7, height=7, pointsize=12) GNOME(display="", width=7, height=7, pointsize=12) \itemdisplaythe display on which the graphics window will appear. The default is to use the value in the user's environment variable DISPLAY. \itemwidththe width of the plotting window in inches. \itemheightthe height of the plotting window in inches. \itempointsizethe default pointsize to be used. http://www.gnome.org and http://www.gtk.org for the GTK+ (GIMP Tool Kit) libraries. Lyndon Drake lyndon@stat.auckland.ac.nz This is still in development state.

The GNOME device is only available when explicitly desired at configure/compile time, see the toplevel ‘INSTALL’ file. x11, Devices. ## Not run: gnome(width=9) ## End(Not run) device
