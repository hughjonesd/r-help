SHLIB {base}R Documentation

Build a DLL for Dynamic Loading

Description

The given source files are first compiled. All specified object files are then linked into a shared library which can be loaded into R using dyn.load().

Usage

Rcmd SHLIB [-o dllname] files

Arguments

files

a list of names of (typically) source files to be compiled and included in the library. You can also include the name of object files which are automagically made from their sources.

dllname

the full name of the shared library to be built, including the extension ‘.dll’. If not given, the name of the library is taken from the first source file.

See Also
