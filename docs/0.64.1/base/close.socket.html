close.socket {base}R Documentation

Close a socket

Description

Closes the socket and frees the space in the file descriptor table. The port may not be freed immediately.

Usage

close.socket(socket)

Arguments

socket

A socket object

Value

logical indicating success or failure

Author(s)

Thomas Lumley

See Also

make.socket, read.socket
