typeof {base}R Documentation

The Type of an Object

Description

typeof provides a means of determining the type or storage mode of an object. typeof returns a character string giving the type of the object.

Usage

typeof(x)

See Also

mode, storage.mode.

Examples

typeof(2)
mode(2)
