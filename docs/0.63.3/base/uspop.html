uspop {base}R Documentation

The Population of the United States

Description

This data sets gives the population of the United States (in millions) as recorded by the decennial census for the period 1790-1970.

Usage

data(uspop)

Format

A time series of 19 values.

Source

McNeil, D. R. (1977). Interactive Data Analysis. New York: Wiley.
