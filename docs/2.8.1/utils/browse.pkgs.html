browse.pkgs {utils}R Documentation

Download R packages

Description

Visual browser for downloading R binary or source packages from a repository, usually invoked from the ‘Packages’ menu of ‘⁠R.app⁠’.

Usage

browse.pkgs(repos = getOption("repos"),
            contriburl = contrib.url(repos, type),
            type = getOption("pkgType"))

Arguments

repos

repository: see install.packages.

contriburl

alternative to repos and type: see install.packages.

type

type of package: see install.packages.

See Also

install.packages
