link.html.help {utils}R Documentation

Update HTML documentation files

Description

Functions to re-create the HTML documentation files to reflect all installed packages. make.packages.html creates the list of packages, and link.html.help is a more verbose wrapper.

Usage

link.html.help(verbose = FALSE, lib.loc = .libPaths())
make.packages.html(lib.loc = .libPaths(), temp = FALSE, verbose = TRUE)

Arguments

verbose

logical. If true, print out a message and use a progress bar.

lib.loc

character vector. List of libraries to be included.

temp

logical: should the package indices be created in a temporary location for use by the HTTP server?

Details

This creates the ‘packages.html’ file, either a temporary copy for use by help.start, or the copy in ‘R_HOME\doc\html’ (for which you will need write permission).

It can be very slow, as all the package ‘DESCRIPTION’ files in all the library trees are read.

For temp = TRUE there is some caching of information, so the file will only be re-created if lib.loc or any of the directories it lists have been changed.

Value

make.packages.html returns a logical showing if it succeeded.
